# YouTube

### Usage

All you should need is the `id` of the YouTube video and you'll get a responsive full width embed.

```html
<YouTube id="YOUTUBE_VIDEOID"/>
```

### Example

```html
<YouTube id="tKjZuykKY1I"/>
```
